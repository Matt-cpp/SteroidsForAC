# SteroidsForAC

A simple open-source cheat made for AssaultCube 1.2.0.2.

It now have a few functionnalities, I plan to add more. I also plan to change the GUI from the terminal to an ImGUI menu.

**Already implemented functionnalites:**
- **Health Hack**: put life to 1000
- **Armor Hack**: put armor to 1000
- **AR ammo Hack**: put assault rifle ammo to 1000

**Functionnalities I plan to add:**
- Aimbot
- Fly
- Rapid fire
- ESP
- No recoil
